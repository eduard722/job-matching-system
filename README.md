# Job Matching System (v1.0)

## 📌 Описание
Система автоматического подбора вакансий с HH.ru на основе rule-based скоринга.

Проект анализирует вакансии, оценивает их по релевантности (навыки, уровень, стек) и отправляет лучшие варианты в Telegram, а также сохраняет результаты в Excel.

---

## ⚙️ Функционал

🔍 Сбор данных
 - Использование HH API
 - Фильтрация по ключевым словам
 - Получение описания вакансий

🧠 Скоринг вакансий
 - Оценка по навыкам (Python, SQL, аналитика и др.)
 - Учет уровня позиции (junior/middle/senior)
 - Бонусы за стек (например Python + SQL)
 - Штрафы за требования опыта

📊 Фильтрация
 - Разделение вакансий на:
 - 🔥 ТОП
 - 🟡 Средние
 - Отсечение нерелевантных вакансий

🤖 Telegram интеграция
 - Отправка вакансий с кнопками
 - Кнопка “Открыть вакансию”
 - Кнопка “👍 Интересно”

🧠 Feedback система
 - Сохранение пользовательских лайков
 - Подготовка к персонализированной рекомендации

💾 Хранение данных
 - sent.json — отправленные вакансии
 - feedback.json — лайки пользователя
 - hh_results.xlsx — выгрузка результатов


---

## 🧠 Как работает скоринг

Итоговый скор рассчитывается как: Score = Skills + Title + Bonus + Stack - Penalty

- Skills — совпадение навыков (Python, SQL и т.д.)  
- Title — совпадение названия вакансии  
- Bonus — стажировки, junior позиции  
- Stack — комбинации (например Python + SQL)  
- Penalty — middle/senior, опыт 3+ лет

## 🚀 Как запустить

1. Установить зависимости: pip install -r requirements.txt

2. Вставить свои данные:
- TELEGRAM_TOKEN  
- TELEGRAM_CHAT_ID  

3. Запустить:

python main.py

 - Поддержка Telegram inline-кнопок
 - Возможность расширения до ML-рекомендательной системы
